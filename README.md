### 🎙️ Real-Time Streaming ASR (Voice AI)
Browser mic audio streams to a FastAPI WebSocket backend in small PCM chunks, returning near-instant transcription — optimized to run on CPU with low latency. Ready to plug into voice agents (STT→LLM→TTS) or telephony pipelines.

AI-powered web app that finds all photos of a specific person from a local image gallery. Uses InsightFace (antelopev2) + ONNX Runtime for 512-D face embeddings, hnswlib for fast similarity search, and FastAPI for the REST layer.

> 📺 [Watch Demo](https://youtu.be/fjN9gNS17Ig)
